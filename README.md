# apps4mind - Live Metabolic Tracker

### Description

A unique application that combines **CBT Psychotherapy**, **Nutrition**, and **React Technology**.
Developed by **Ro»ôu Adrian Francois**, this tool helps users visualize weight loss in real-time,
transforming the hunger sensation into a positive signal of metabolic progress.

### Key Features

- **Live Burn Rate:** Real-time visualization of calories burned per second.
- **CBT Panic Button:** Emergency psychological intervention for intense cravings.
- **Dynamic Visualization:** Color-coded graph (Red for storage, Green for active fat loss).
- **Privacy:** Data is persisted locally via LocalStorage.

### Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Recharts (Data Visualization)
- Lucide React (Iconography)
